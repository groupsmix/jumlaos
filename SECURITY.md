# Security Policy

## Reporting a vulnerability

Email **security@jumlaos.ma** with details. Please do not open public GitHub issues for security reports.

We respond within 72 hours and aim to ship a fix within 30 days for critical findings. Coordinated disclosure is appreciated.

## Supported versions

| Version    | Supported          |
| ---------- | ------------------ |
| 0.1.x-beta | yes (active beta)  |
| < 0.1      | no                 |

See `docs/security.md` for the threat model and `docs/security-checklist.md` for the pre-release checklist.
